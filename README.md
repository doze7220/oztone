# OZtone

将来のタスクバー放置ゲーム開発に向けたテストケースとして作成する、自分専用のMP3プレイヤーです。

## コンセプト
* C++とWin32 APIで作成する、デスクトップアクセサリ風のMP3プレイヤー。
* ウィンドウ枠を持たず、一部UI以外はドラッグ不可というストイックな設計。
* 将来のゲーム開発を見据えた「枠なし・半透明ウィンドウ」の技術検証を兼ねる。

## 主な機能
* MP3ファイルの再生。
* UI上の「ドロップアイコン」にファイルをドラッグしてプレイリストを作成。
* 再生方法は「常時シャッフル・リピート再生」のみ。
* Windowsのメディアキー操作に対応。
* プレイリストはユーザーフォルダに生成・保存され（パス設定可能）、起動時やリフレッシュ時に読み込まれる。

## UI・画面構成
* **基本サイズ:** デフォルトで 1024x512。
* **背景とビジュアル:** 表示領域全面に再生中のアルバムアートをフィット・トリミングして表示し、さらにその上にビジュアライザを重ねて表示する。
* **テキスト（左上）:** 大きめに曲名、アーティストを表示。
* **サブ画像（左下）:** アルバムアート全体を表示（透過なし、半透明、フェードアウトをコンフィグで設定可能）。
* **再生リスト（右）:** 半透明で表示。リストにはアルバムアート縮小アイコン、タイトル、アーティスト、再生時間が並ぶ。
* **シークバー（下部）:** 下部50pxぐらいに表示するが、操作はできず表示のみ。
* **操作ボタン:** 「再生（停止）、前の曲、次の曲」だけ。

## 開発環境・技術スタック
* 言語: C++
* フレームワーク: Win32 API
* AI-IDE: GoogleAntigravity
* コンパイラ: Build Tools for Visual Studio (MSVC)
