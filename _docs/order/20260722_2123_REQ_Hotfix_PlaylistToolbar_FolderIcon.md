###### 【実装要件】
本タスクでは、プレイリスト一覧モード時に左端の「上の階層へ」ボタン（フォルダアイコン等）が空白（非表示）になっている状態を修正し、エクスプローラーのように普遍的なUIとしてグレーアウト表示させる。

*   `src/Widget_Playlist_Toolbar.cpp` のツールバーアイコン描画処理を調査し、`ctx.isPlaylistListViewMode` が `true` の場合にインデックス0（上の階層へ）のアイコン描画をスキップしている箇所を特定する。
*   スキップ処理を削除し、代わりに描画ブラシの不透明度を下げる（例: `0.3f` や `0.4f` など）ことで、無効状態（グレーアウト）としてアイコンを描画するように修正すること。
*   ホバー時の説明テキスト（ツールチップ）についても、グレーアウト中は表示しない（または無効を示す）などの適切な対処が含まれているか確認し、違和感のないUXとすること。
