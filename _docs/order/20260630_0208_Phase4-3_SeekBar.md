【AIへの指示（Phase 4 - Step 3 : シークバーと時間表示の実装）】
現在のプロジェクトをベースに、次の実装を開始します。

**【今回の目的：Phase 4 - Step 3】**
AudioPlayerから取得した再生時間を利用し、Direct2D/DirectWriteを用いてUI上にシークバーと現在時間（mm:ss / mm:ss）を描画します。見た目の実装とデータ連携を一挙に行います。

**【実装要件】**
1. **ConfigManager の拡張（シークバー設定）**:
   - `[Layout_SeekBar]` セクションを追加し、以下の値を読み書きできるようにしてください。
     - `WidthRatio`: 0.95 (float, ウィンドウ幅に対する全体幅の割合)
     - `Height`: 3 (int, バーの太さpx)
     - `BottomOffset`: 50 (int, 画面下からのオフセットpx)
     - `BgOpacity`: 0.3 (float, シークバー下地の不透明度)
     - `TimeFontFamily`: "Consolas" (string, 等幅フォントを指定)
     - `TimeFontSize`: 12.0 (float, フォントサイズ)
     - `TimeAreaWidth`: 100 (int, 時間テキスト描画専用の固定領域幅px)

2. **Application の拡張（データの計算とフォーマット）**:
   - `Application::Run` または描画ループ内で、`AudioPlayer` から現在時間と総時間を取得してください。
   - 進行度 (`progress`) を 0.0f 〜 1.0f の範囲で計算してください（総時間が0以下の場合はゼロ除算を防止し 0.0f とする）。
   - 取得した時間を元に、0埋めなしの「mm:ss / mm:ss」フォーマットの文字列 (`std::wstring`) を生成してください。
   - `Renderer::Render` の引数に、計算した `progress` と `timeString` を追加して渡すように変更してください。

3. **Renderer の拡張（描画処理）**:
   - `Render` メソッドの引数を `void Render(bool isHovered, float progress, const std::wstring& timeString)` に変更してください。
   - 背景やロゴ等の描画の後（一番上）にシークバーを描画します。以下のロジックで論理座標[2]を計算してください。
     - `総幅 = ウィンドウ論理幅 * WidthRatio`
     - `左端X = (ウィンドウ論理幅 - 総幅) / 2.0f` (中央寄せ)
     - `バー領域幅 = 総幅 - TimeAreaWidth`
   - **バーの描画**:
     - 左端X から `バー領域幅` 分の長さに、不透明度 `BgOpacity` の白色で下地の矩形を描画します。
     - その上に、左端X から `バー領域幅 * progress` 分の長さに、不透明の白色で実線の矩形を描画します（高さは `Height`）。
   - **テキストの描画**:
     - `IDWriteTextFormat` を `TimeFontFamily` と `TimeFontSize` を用いて作成し、テキストの配置を「右寄せ（DWRITE_TEXT_ALIGNMENT_TRAILING）」、「上下中央揃え（DWRITE_PARAGRAPH_ALIGNMENT_CENTER）」に設定します（パフォーマンスのため、TextFormatはメンバ変数として保持・再利用することを推奨）。
     - 描画矩形を `[左端X + バー領域幅, Y座標, 左端X + 総幅, Y座標 + Height]` とし、作成した時間文字列を描画します。
