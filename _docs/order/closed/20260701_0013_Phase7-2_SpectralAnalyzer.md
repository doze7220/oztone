   - 直近の修正である「色決定のX座標ベース7等分」「上限 maxHz = 11000.0f 付近への引き下げ」、および「WAVE_STEP_X (3.0f) による間引き」「落下減衰（重力スムージング）」「ネオンの二重描画（HOLLOW / OPEN）」は絶対に崩さず維持してください。
