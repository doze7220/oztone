#### 【実装要件】
プレイリストのツールバーにおいて、テキストの左寄せ化、ホバー時のトラック数非表示、およびアイコンとテキストへの左マージン(2.0f)の追加を実装する。

*   **要件1: ツールバーテキストの左寄せ化 (`src/Widget_Playlist_Resources.cpp`)**
    *   `m_toolbarTextFormat` のアライメント設定を確実に `DWRITE_TEXT_ALIGNMENT_LEADING`（左寄せ）へと修正する（もし `CENTER` が残っていれば上書き・削除すること）。

*   **要件2: アイコンおよびテキストへの左マージン追加 (`src/Widget_Playlist_Toolbar.cpp`)**
    *   ツールバー上段のアイコンを描画するループ等において、描画の基準となる開始X座標に `+ 2.0f` のマージンを加算する。
    *   ツールバー下段の左寄せ説明テキスト（プレイリスト名やツールチップ等）を描画する際の矩形（`textRect` の `left` 等）にも、アイコンの左端ラインとピッタリ揃うように `+ 2.0f` のマージンを加算する。
    *   ※この2pxの調整は絶対的なレイアウト比率であるため、Configには追加せず、ソースコード上に `2.0f` としてハードコードすること。

*   **要件3: ホバー時のトラック数（総曲数）非表示化 (`src/Widget_Playlist_Toolbar.cpp`)**
    *   ツールバー下段の右端にトラック数（例: `120 Tracks`）を描画している処理を見つけ、その処理全体を `if (ctx.playlistToolbarHoveredIndex == -1)` の条件ブロックで囲む。
    *   これにより、どのアイコンにもホバーしていない通常状態でのみトラック数が表示され、ホバー時（説明テキスト表示時）には非表示となって文字の激突を防止する。
