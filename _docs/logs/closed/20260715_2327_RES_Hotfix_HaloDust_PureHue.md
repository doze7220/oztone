# RES:HOTFIX作業レポート: Halo Dust 純粋なHUEカラー生成のテスト

## 1. 実装目的
Halo Dustビジュアライザの色生成において、既存の補正処理をスキップし、ハッシュ値から純粋なHUE（彩度100%、明度100%）を生成して適用するテスト実装を行う。

## 2. 調査内容
`src/Visualizer_HaloDust.cpp` の色生成部分を改修する。ハッシュ値から直接HUEを求め、自作の `HsvToRgb` ヘルパー関数を通して純色（彩度1.0、明度1.0）を取得し、既存の加算合成や青・緑シフトを回避してそのままベースカラーとして適用する。

## 3. 対象ファイル
* `src/Visualizer_HaloDust.cpp`

## 4. 実装タスクリスト
- [x] タスク1: HSV to RGB 変換ヘルパー関数の追加
- [x] タスク2: ハッシュ値からのHUE生成と純色適用の改修

## 5. 詳細作業内容
* タスク1: HSV to RGB 変換ヘルパー関数の追加
    - `src/Visualizer_HaloDust.cpp` の無名名前空間内に、指定されたシグネチャの `HsvToRgb` 関数を実装し追加した。
* タスク2: ハッシュ値からのHUE生成と純色適用の改修
    - `Draw` メソッド内で色を決定するロジックを改修した（元の加算合成やカラーシフト計算部分は置換により削除）。ハッシュ値から直接HUE角度を算出し、`HsvToRgb` で完全な純色を取得。これをそのまま最終的なRGB要素として適用した。
