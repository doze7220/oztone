# 作業レポート (RES)

## タスク概要
- **タスク名**: Hotfix / 仮想スクロールバグ修正3 (メタタグ抽出の二重加算解消)
- **対象ファイル**: `src/Application_Initialize.cpp`

## 原因
仮想スクロール（マウスホイール操作）時のドラム回転において、メタデータを供給する `dataProvider` ラムダ式内で、相対距離（`relDist`）を加算する基準点として誤った変数が参照され得る構造になっており、結果としてテキストメタデータが実際の曲とズレる（二重加算に類する）バグを引き起こしていました。

## 対応内容
- `src/Application_Initialize.cpp` の仮想スクロールコールバック内にある `dataProvider` ラムダ式を修正しました。
- データ取得対象のインデックスを計算する基準（ベース）を、必ず `m_playlistManager.GetCurrentIndex()` を直接参照するように修正しました。
- 取得先インデックスの計算式を `(m_playlistManager.GetCurrentIndex() + relDist + total) % total` の形へ純化し、他の変数への依存を排除しました。
