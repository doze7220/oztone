# Phase 3 Step 1 実装レポート

## 実装目的
ウィンドウの完全枠なし化（アクセサリ化）と、左上のアプリアイコン領域を利用した「ドラッグ移動」および「仮の右クリック終了機能」の実装。

## 変更ファイル
- `src/Window.h`
  - `IsInLogoRegion` メソッドの宣言を追加
- `src/Window.cpp`
  - `IsInLogoRegion` メソッドの実装を追加
  - `WindowProc` にて `WM_LBUTTONDOWN` を捕捉し、ロゴ領域内でのクリック時に `ReleaseCapture()` と `SendMessage(hwnd, WM_NCLBUTTONDOWN, HTCAPTION, 0)` を発行しドラッグ移動を実装
  - `WindowProc` にて `WM_RBUTTONDOWN` を捕捉し、ロゴ領域内でのクリック時に `PostMessage(hwnd, WM_CLOSE, 0, 0)` を発行しアプリ終了を実装
- `PROJECT_ARCHITECTURE.md`
  - Phase 3 Step 1 完了時点として全体フローやクラスリファレンスの記載を更新

## 実装内容
- 初回起動時から枠なし（`WS_POPUP`）にするための修正は、`ConfigManager.cpp`のコンストラクタによる初期値が全て `false` (0) となっていることと、`Window.cpp` のスタイル設定で両フラグが false の際に `WS_POPUP` が付与されるロジックにより、既存実装のままで要件を満たしていることを確認しました。
- `WindowProc`内をクリーンに保つため、座標判定処理を `IsInLogoRegion` メソッドへ分離して実装しました。

## ビルドテスト結果
- 警告・エラーなくコンパイルが通り、ビルド成功を確認しました。

## 懸念点・申し送り事項
